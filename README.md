# Diseño Responsivo Banco Pichincha

## Descripción

Aplicación Android desarrollada en Android Studio que replica la pantalla de inicio de sesión de la aplicación móvil Banco Pichincha utilizando XML y componentes responsivos.

## Tecnologías Utilizadas

* Android Studio
* Java
* XML
* ConstraintLayout
* ScrollView
* CardView

## Características

* Diseño responsivo.
* Logo y encabezado de Banco Pichincha.
* Ilustración principal.
* Opciones de inicio de sesión.
* Accesos rápidos: Ubícanos, Clave Digital y Llámanos.
